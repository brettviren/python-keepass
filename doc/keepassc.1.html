NAME

keepassc - command line interface for KeePass v4 (.kdbx) files  

SYNOPSIS


 search [options] <key> <dbfile> [...]
 dump [options] <dbfile> [...]  

COMMANDS

Several commands and options are available.  

search [options] <key> <dbfile> [...]

Searches given key in all entries in keepass databases. The search is case insensitive and ignores deleted entries in the backup folder. For each entry the folder name, title and notes are searched.
-p, --show-passwords
Also show passwords of entries.
 

dump [options] <dbfile> [...]

Pretty print contents of keepass databases. Output is the underlying XML structure.
-p, --show-passwords
Show passwords in database dump.
 

HELP OPTION

Specifying the help option displays help for keepassc itself, or a command.
For example:
  keepassc --help - display help for keepassc
  keepassc search --help - display help for the search command
